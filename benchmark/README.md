## Success Criteria
1. **Product Integrity**: Does the product look like the original?
2. **Platform Fit**: Does it match Shopee/Lazada/XHS aesthetics?
3. **Realism**: Does it look like a real photo vs AI-generated?
4. **Text Safety**: Are logos and labels preserved?
